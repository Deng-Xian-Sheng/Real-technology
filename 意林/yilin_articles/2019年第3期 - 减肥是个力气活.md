# 减肥是个力气活

关于减肥的话题，每过一段时间就会卷土重来。

年初聚会，几个大胖子表兄妹们互相激励着一起减肥，还定好了奖惩制度，成功者获奖1万，从失败者的押金里扣。参与比赛的都交了押金，几个不太肥的也乐呵呵观战。今年长假又聚，发现有个大胖子活活减下来20斤！这引发了更大面积的焦虑，连那些不太肥的都恐慌了，因为马上就没有比你更肥的垫底了，你有可能会成为那个最肥的了！

我妈更可气，仿佛实话实说就像对命运交了底儿，会被命运偷袭似的，聚会上一直昧着良心夸我表妹：“减肥减得人都漂亮了，轮廓清晰了，线条出来了，不像某某（就是我），看看现在胖成个啥了？”

一边说，一边又递给我一个包子。我正襟危坐表示，会虚心听取表妹传授的减肥秘籍，其实内心更想听的是她怎么做香辣蹄花。表妹从前开私房菜餐厅，可她如今皈依了减肥教，想必和这些美食成了死对头，就不好再提了。

表妹给我们发了减肥食谱，把我拉进了减肥群，成员天天发一日三餐，互相监督。食谱里规定，早晚餐都不能有碳水化合物，所有淀粉类的，诸如土豆、红薯、莲藕、山药什么的也不可以蒙混过关。蔬菜、酸奶、木耳这些只能挑两样吃，往饱吃。我试了，只吃青菜木耳，还没吃饱就吃腻了；胃口不好的，吃上几次就吃伤了。两样往饱里吃，其实就是让你不要吃饱。这个制定减肥食谱的人简直居心叵测：让你对食物的兴趣降到最低，办法就是把食物变难吃，令人吃到烦，吃到没胃口。

依據减肥餐吃到第三天，的确感觉身体轻松了，但并没有传说中的愉悦感，相反，变得不那么愉悦了。凛冬将至，饭都吃不饱，愉悦个啥？

再坚持了几天，更多的负面情绪出来了：心火大、死心眼、发脾气还没劲儿发……再这么下去恐怕要抑郁了。尽管人们“安利”说减肥是个技术活，可事实会告诉你，减肥其实是个力气活，需要用尽全身力气对抗食欲，直到你再看到主食和肉类，就跟看到毒药一样，对很多食物也变得不友好。

我感到很愧疚，毕竟食物并没有像我这么小心眼儿。尤其是饥肠辘辘的冬夜，走在回家的路上，看到炸酥肉、烤鱿鱼、炒毛栗子……闪耀着黄金色泽的食物，它们不计前嫌，依旧用馥郁的香气热情地招呼我：来吃呀，干吗不呢？

放弃减肥之前，我给自己找了不少理由：比如，类似“身体都管理不好，还能管理好人生吗”这种话，一旦成为不容置疑的真理，就变得很可疑：管理身体，管理情绪，是想把自己管成整齐划一完美无缺的机器人吗？这难道不是人类面对机器人时代的集体焦虑感作祟吗？此外，我还融合了身体美学方面的思考：那些痴迷于减肥的人，怕是生活中没别的可坚持了，不如折腾自己身体吧……

好吧，我承认，减肥还是有用的：通过减肥，更加刺激了我对美食的欲望。

# 开怀篇（1）

**气质**

刘备对赵云说：“子龙啊，你看云长和翼德与大哥都结拜了，你也加入我们吧！”赵云想了想，说：“还是不要了，我觉得叫赵四不符合我的气质。” 

**叫什么**

去图书馆，坐了一会儿一小帅哥（就算帅哥吧，我也没仔细看）过来跟我搭讪，问我：“同学，你叫什么啊？”当时我一脑抽说：“我没叫啊。” 

**辣眼**

让我妈给我弄两片土豆我敷眼睛，后来实在辣得我不行，一直以为是不是淀粉起了什么反应，敷了好一会儿实在受不了了，问我妈：“怎么这么辣眼啊？”她告訴我她切了辣椒忘洗刀了…… 

**中奖**

刚才走在路上，接了一个陌生电话，一个女的开口就说：“你好！恭喜你中了我们公司二等奖30万元！”我还没说话，她自己哈哈大笑，说：“不好意思，第一次骗人，没忍住……”

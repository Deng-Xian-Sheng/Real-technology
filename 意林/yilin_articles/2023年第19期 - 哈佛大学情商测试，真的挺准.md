现在，请静下心来，诚实地回答下面的测题。

### 心理测试开始：

1.坐飞机时，飞机突然受到很大的震动，机身开始左右摇摆。这时候，您会怎样？

A.继续读书或看电影

B.注意事态的变化，以防万一

C.A和B都有一点

D.根本没注意到

2.带一群4岁的孩子去公园玩，其中一个孩子由于别人都不和他玩而大哭起来。这个时候，您会怎么办呢？

A.让孩子们自己处理

B.帮助他想办法

C.轻轻地告诉他不要哭

D.转移这个孩子的注意力

3.假设您是一名大学生，想在某门课程上得优秀，但是在期中考试时只得了及格。这时候，您该怎么办呢？

A.制订一个详细的学习计划

B.决心以后好好学习

C.考不好没什么大不了的，把精力集中在其他考得好的课程上

D.去拜访任课教授，试图让他给您高一点的分数

4.假设您是一个保险推销员，去拜访潜在客户，他们对您很敷衍。这时候，您会怎么做？

A.希望明天会有好运气

B.考虑自己是否适合做推销员

C.在下一次拜访时再做努力

D.考虑去争取其他顾客

5.假如您是一个经理，提倡在公司中不搞种族歧视。一天您偶然听到有人在开有关种族歧视的玩笑。您会怎么办呢？

A.不理它，只是玩笑而已

B.把那人叫到办公室斥责一顿

C.当场告诉他，这种玩笑是不恰当的

D.建议开玩笑的人去参加一个有关反对种族歧视的培训班

### 答案：

1\. 除了D 以外的任何一个答案都不错，选择答案D反映了您在面临压力时经常缺少警觉性。

A = 20 ，B = 20 ，C =20 ，D = 0

2\. B是最好的选择，情商高的父母善于利用孩子情绪状态不好的时机对孩子进行情绪教育，帮助孩子明白是什么使他们感到不安，他们正在感受的情绪状态是怎样的，以及他们能进行的选择。

A = 0 ，B = 20 ，C = 0，D = 0

3\. A，自我激励的一个标志是能制订一个克服障碍和挫折的计划，并严格执行它。

A = 20 ，B = 0 ，C = 20，D = 0

4\. C为最佳答案，情商高的一个标志是面对挫折时，能把它看成一种可以从中学到东西的挑战，坚持下去，尝试新的方法，而不是放弃努力，怨天尤人，变得萎靡不振。

A = 0 ，B = 0 ，C = 20，D = 0

5\. C，形成一种欢迎多样化的气氛的最有效的方法是公开挑明这一点，当有人违反时，明确告诉他您的组织的规范不容许这种情况发生。不是力图改变这种偏见（这是一项更困难的任务），而只是让人们遵照规范去行事。

A = 0 ，B = 0 ，C = 20，D = 0

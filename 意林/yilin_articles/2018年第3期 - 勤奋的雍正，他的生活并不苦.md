# 勤奋的雍正，他的生活并不苦

**圆明园里有雍正的诗和远方**

和儿子乾隆喜欢到全国各地游山玩水不同，雍正大多数时间都宅在圆明园。作为圆明园的总设计师，雍正的设计理念就是建造一個山清水秀、鸟语花香、亭台楼阁、有诗情画意的精神家园。 

雍正生性低调喜静，圆明园建好后，他就搬到这里，将其当做办公、生活的主要场所。 

在不知情的人眼里雍正是个工作狂，但是他骨子里却有一种文艺气息，春天里踏青赏花，夏日里竞舟纳凉，秋天里望月观菊，冬日里参禅听雪。 

而且每次的休闲活动一定要让御用画师郎世宁画下来，他再配上诗转发。若不信，看看现存的《雍正十二月行乐图》就明白了。 

在行乐图中，他时而穿着汉服，扮清流濯足的村夫，过悠闲自在的田园生活；时而又着蒙古袍，摆出一副弯弓射大雕的样子；时而又化身苏东坡在山野里题壁。 

最令人捧腹大笑的是他还赶时髦，穿上西洋袍，系上围巾，再戴上豪华的假发，体验了一把欧洲贵族的生活，并让郎世宁画下来，慢慢地欣赏，自我陶醉。 

没有一颗自恋的心，没有一份热爱生活的情趣，又怎能去体验这多姿多彩的生活？ 

**对细节有着狂热追求的设计师**

他有两只爱犬，一只叫造化狗，一只叫百福狗。话说这两只狗的名儿取得真接地气，得有多大的造化和福气才能深得皇上的宠爱？而且雍正屡屡传旨给皇家服装与工艺设计制造院——造办处，让他们按照自己的设计给狗做套头衫、狗笼、狗垫等。 

他下令给造化狗做麒麟衣，还为其设计了一款老虎式纺丝面软里子的套头衫。做好以后，结果狗狗穿上耳朵窝在衣服里，很不舒服，又让人加上了两只耳朵。 

从圆明园的总设计师到打扮宠物狗，看似不可思议，其实源于他对设计的狂热追求。 

在记载中，雍正喜欢用当时进口的装潢新材料——玻璃来进行居室的装修。如圆明园后殿壁灯以及双圆玻璃窗就是他的作品。 

壁灯容易理解，看看他设计的双圆玻璃窗。为了克服空间狭小的特点，他充分利用玻璃的透明特点，在墙上安装一个双面的圆窗，朝向室内的玻璃上画“节节见喜”图案，另一面玻璃则无。但在窗后板墙上与圆窗相对应的地方也画上一模一样的“节节见喜”图案。 

人在仙楼内，就会看到前扇玻璃与后面板墙上两重图样隐约叠在一起，从而获得视觉上的趣味，也产生进深较长的错觉。 

这是现代家庭装修设计中常见的做法。但在当时，西洋玻璃刚引进国内，雍正既利用了新材料透光的特点，又考虑到中国传统园林的艺术风格，将二者有机地结合，还是颇具设计天赋的。 

造办处档案还记录了雍正设想出一组“搁炉小香几”的新样式。据说这是一个方六七寸、高二三寸的袖珍香几，面上仅仅放置一个小香炉，插放香匙、香箸的瓶子被取消，由四个支足的中空孔道作为匙、箸的插管，香盒则隐藏在几面下的抽屉里。 

除此以外，他还喜欢琢磨鼻烟壶以及盛花香器的设计。 

他曾利用当时的奢侈品——象牙，来设计雕花香盒或香袋，装饰房间。这等低调奢华的生活情趣，非普通人能享受得起。 

这些小事，让我们看到一个性格更加立体多面的雍正，他并非是那个生活忙碌的工作狂，而是一个懂得诗情画意，对精致生活有执著追求的设计师。

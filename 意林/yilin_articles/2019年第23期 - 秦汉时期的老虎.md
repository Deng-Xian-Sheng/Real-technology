# 秦汉时期的老虎

最近，国家博物馆“小城故事：湖南龙山里耶秦简文化展”中展出了五枚简，记录了一个秦朝时期“买虎肉”的事。文字为：“卅五年十月壬辰朔乙酉，少内守履出黔首所得虎肉二斗，卖于更戍士五城父□里阳，所取钱卌，率之斗廿钱。令史就视平。魋手。”

其实这些内容不是在讲故事，而是为了留下一个收支凭证，让买卖流程规范，有监督人员、记录人员，记录的格式规整。里耶秦简的内容主要就是日常行政的记录。

看到这样的文物，很多人对当时老虎的生存情况有浓厚兴趣。上述简文中的老虎是被普通百姓抓住的，而这只老虎的遭遇也并非特例。里耶秦简里还有记载都乡守向上级汇报抓到老虎的事情：“廿八年五月己亥朔甲寅，都乡守敬敢言之：得虎，当复者六人，人一牒，署复□于从事，敢言之。五月甲寅旦，佐宣行廷。”说的就是，就因为抓到了一只大老虎，六个人得以免除徭役。

捕到老虎有奖励，不是仅见于里耶简，傳世文献中也有相关内容。《尔雅.释兽》郭璞注引律：“捕虎一，购钱三千；其豿（狗）半之。”但这个和出土文献的记载是有差异的，因为它是给予捕虎者以赏金，而出土文献中则是免除徭役。法史学者程树德认为这一条法律“盖晋律文也。《集韵》引作汉律，盖晋与汉同”。可能是随着时代发展，相关制度有了变化。郴州晋简还有提到“口廿三捕虎工”“口五十捕虎工”，意思就是有很多人是专业的捕虎工。说明到了西晋，捕虎这个事情更加专业化了。

如果只看史书记载，我们就会推断，当时人们捕虎的目的可能是为了占有珍禽异兽。比如西汉文帝时上林苑有“虎圈啬夫”这一职务；汉成帝命右扶风发民入南山，捕熊罴豪猪，虎豹狐兔等。因为现在能够见到出土文献，就知道抓到老虎居然还可以用来卖虎肉。

不过，动物保护人士大可不必为老虎出头。因为在当时，老虎这样的猛兽可能常常会出现在人类活动的区域，造成虎患。之所以捕虎，可能还是要为民除害。

到东汉的时候，地方循吏还经常处理和虎患相关的事情。《后汉书·循吏列传》中记载童恢在地方任职，民众为虎所害，于是设下陷阱捕到了两只老虎，童恢还对老虎进行了教育。对它们讲：“天生万物，唯人为贵。虎狼当食六畜，而残暴于人。王法杀人者伤，伤人则论法。汝若是杀人者，当垂头服罪；自知非者，当号呼称冤。”意思是你们两个当中，杀人的那只你就低头认罪，没有做过的你就喊冤。

两只老虎听了，一只低头闭目，一只鸣吼、踊跃自奋。结果可想而知，认罪的那只被杀了。

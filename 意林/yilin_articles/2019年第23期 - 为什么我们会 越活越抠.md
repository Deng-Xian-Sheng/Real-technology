# 为什么我们会 越活越抠

上周末生日，跟大姚约好去市中心一起吃饭。大姚是我助理的姐姐，一个大学刚毕业没几年的社会新人，我经常在她身上看到七八年前年轻时候自己的影子。二十五六岁女生的烦恼无非就是，减肥不成功，没有男朋友，钱不够花。三大烦恼贯穿生活，不过喝一杯奶茶又原地复活了。

大姚说，你别开车了吧，过生日喝一杯。

我想了想，打车来回要二百，不禁揪心地疼痛，出门情不自禁还是开上了车。

到达目的地和大姚会合，忍不住问了一句：你是坐地铁吗？你怎么那么快？

她脸上浮起一种莫名其妙的笑容，说：哎呀，这天这么热，我不想坐地铁了，我打车来的。

多少钱？

八十。

哇，你真舍得花钱。

说这句话是因为我琢磨了下，大姚作为外企员工，收入大概是我的十分之一。

大姚沉默了两秒钟，说：可是天真的很热好吗？你怎么跟我老板一样，我老板也问我，你每天怎么上班。

你怎么上班？

打车啊。

对她来说，打车上班好像就是一种最正当的需求，除了一点，早高峰期间总是要排队，排队15分钟以上，有迟到的风险。

我哀叹了一声，我怎么这么想不开？过生日还舍不得两百。

大姚安慰我，我老板比你还省呢，他骑电瓶车上班。

什么？你老板不是大中华区的leader吗？他骑电瓶车？

对呀，他家就一辆车，他说给老婆开了，你知道的，上海男人……

我一时有点风中凌乱，有点三十年河东三十年河西之感。几年前我还经常大骂我妈，不舍得花钱，老是买点便宜东西，喜欢跟着小区里的大妈们，一起去抢购一百块三条的连衣裙，88块古镇特价游，只要哪里送米送油，哪里就有她們的身影。

人的消费习惯，好像跟口味一样，都是儿时训练出来的。我有一次跟最后一班地铁擦肩而过，因为舍不得一百多元的打车费，终于在麦当劳里买了杯咖啡，和朋友一起坐了通宵。

我猜大姚要是听到这种故事，肯定跟我听沈女士忆苦思甜一样，内心肯定在翻白眼：以前这些老皇历就不要提了。

最挫败的一层是，我终于也来到了对自己抠，对小孩大方的年纪。像无数个中年人一样，给小孩掏学费义无反顾，给自己花钱会先想想，有必要吗？好像也不是很有。

不要瞧不起那些衣着朴素的中年人，他们不是抠，只是没有了给自己花钱的欲望。

大中华区的leader可以骑电瓶车上班，一个住别墅的朋友某天穿着她的家居外套说，这是我从别人身上买的，你看，是不是喂奶超方便？我看着那件黑乎乎毛茸茸的袍子，重重点了点头。微博上500多亿市值的上市总裁坦白说，现实中的霸道总裁根本不是《欢乐颂》里小包总那样，穿着定制西服出门扭胯飞媚眼。其实也是一身优衣库无印良品瘫在办公室里。

我跟大姚在巴黎的时候，一人置办了一身maxmara大衣，买的时候她在店里像在思考人生大事一样犹豫着，到底要不要买一件两万块的大衣？好像真的有点太贵了，万一亲生父亲不报销怎么办？我一边喝着美丽的导购员送来的香槟，一边说：不贵不贵，毕竟是maxmara啊。

当时我想，都去法国了，能不买吗？

回来后小陈说儿子要交一万多的英语补习班费，我本来心不甘情不愿，心想交一万多他就能学好。后来看了看身上的大衣，一件大衣小孩能学一年的英语。我到底为什么要买这么贵的大衣？值得吗？

不值得。

中年人花每一分钱，都会自然而然换算成小孩的补习班钱。一双鞋两千，那三双鞋就是一个高尔夫球营队；一件真丝衬衫两千，那四件衣服他就能去趟三亚学冲浪；一条项链两万多，小孩能买一整个屋子的乐高嘛。

我终于彻底理解了那些只在房和车上花大钱的中年老父老母。

生活的琐碎彻底击碎了我们，毛姆写《月亮与六便士》，说满地都是六便士，他却看到了月亮。

中年人都是忙着捡六便士的人，人生大纲拿出来：能省一分是一分。

我问起大姚的maxmara大衣，她说真的很值，幸亏买了。我想了想我的大衣只出场过两三次，大部分时间，就跟那个要喂奶的朋友一样，蓬头垢面随便穿穿。

年轻人总是把自己打扮得漂漂亮亮，天天都在看着天上的月亮，有人赚到第一个一百万就买一辆玛莎拉蒂，有人月入一万就敢买一只香奈儿，有人毕业就打算环游世界。

就算花光最后一分钱，那又怎么样？月亮好看就行了。

收房后，我家里来了两个设计师，差不多资历，差不多年龄，差不多报价。唯一的差别是，一个刚刚结婚生完小孩，才几个月大。一个二十大几，尚未婚配。两人都有买车的需求，在一起聊起同一款车，刚生完小孩的，斩钉截铁地说，太贵了。未婚男青年笑嘻嘻说：还可以吧，我打算买了。呵呵，年轻人，你对生活的考验一无所知。

最终你会抠得像只铁公鸡，甚至会为自己过去的虚荣感到震惊，居然能瞎花这么多钱。

大姚总结发言：那我还没结婚呢，我不能过得跟你一样。

但生活，总会把你，一个看月亮的人，逼成捡六便士的。

我希望有人站出来告诉我，他不会，他没有。

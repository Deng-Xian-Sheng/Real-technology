# 一杯敬C罗，一杯敬C位

不知道球迷们看不看《创造101》，也不知道《创造101》的各位小姐姐看不看世界杯。我都看了，各有各的精彩，姑娘们香喷喷亮晶晶好听又好看，球场上的帅哥们大汗淋漓时不时还血洒绿茵也特别燃。C罗和梅西，还有内马尔、羅伊斯、萨拉赫、阿扎尔……跟小姐姐们一样，他们也各有来自全世界的粉丝，当然也都会听到不同的声音。即使火为C罗，也曾被郝海东在2010年断言“火不过三年”。所以十几二十岁的《创造101》的小姐姐们也不用在意那些diss你们唱歌跑调、不配出道、谁谁谁比谁谁谁强多了之类的言论。倒是在世界杯的小哥哥身上有很多值得你们学习的东西。

就说C罗吧，作为一名每场比赛被全世界的对手重点盯防的中锋，今年33岁的他其实已经过了理论上的前锋队员的巅峰时期。常年被“重点照顾”以致受点儿小伤对他来说是家常便饭，按照这个势头，当年郝海东说“火不过三年”也不是毫无根据的。而且他太红了，在欧洲顶级俱乐部踢球，受到全世界球迷的仰望，不管是商业上的诱惑还是心理上的膨胀都是一名球员必经的“修行”。但本届世界杯上看C罗，不管是身材管理还是运动爆发力，都还处于几近完美的状态。葡萄牙队医曾透露C罗的“保养秘诀”：他是一个注重一切的球员，会保证饮食、睡眠以及心态上的完美。为了延长自己的运动寿命，C罗私下里还是业余学医小狂人。再说今年夏天最时髦的词：C位（最中间位置）。《创造101》的发起人黄子韬说：女团的成功靠的是什么？勤奋和能力。导师胡彦斌也说过：现在女团太多了。作为女团成员，有天赋拼天赋，有拼搏靠拼搏，有努力靠努力。什么都没有就只能被淘汰。冠军孟美岐无疑是《创造101》女孩中最具备全面偶像素质的选手。她同时拥有胡彦斌说的天赋、拼搏和努力。不到20岁的女孩子表现出来的除了唱歌跳舞的绝对实力，在节目中她会帮助不擅长跳舞的组员一点点地抠动作，也会把自己在韩国做练习生这几年学到的技巧倾囊相授。她是一位实力优秀的选手，也是一个有责任心的队长。优秀不是天生的，也不是一夜长成的，孟美岐14岁就独自去韩国成为一名练习生，语言不通，没有父母的陪伴。经过残酷的淘汰和训练，孟美岐在2005年终于作为宇宙少女的一员成功出道。和训练生伙伴们十几个人蜗居在一个不足四十平方米的宿舍里。

不管是综合实力强劲的孟美岐和吴宣仪，还是有个性的Yamy、有争议的杨超越，出道不是大获全胜的结束，而是前无古人后路未知的女团之路的开始。不管是被称赞还是被诋毁的，前途都是要看她们以后的作品和是否能够获得成长。刚刚整个世界杯身价最高的法国队闷平丹麦队的比赛收获了满场嘘声，两队联手苟且地小组出线了，但观众却把掌声献给了同组被淘汰却认真对待比赛和观众的秘鲁队和澳大利亚队。

这就是为你们疯狂打call的观众，你是身价不菲的豪门球队，但当你对不起你的荣誉时，荣誉也会抛弃你。那些出道或者没出道的小姐姐，离C位有多远当然重要，更重要的是你对待娱乐圈的态度。《创造101》结束后，《明日之子》第二季将火速续档。作为同样的造星节目，第一季朴实平凡的毛不易却凭借其独特的个性魅力与《消愁》的现象级流行一举夺冠。

一杯敬毛不易，一杯敬孟美岐。在这个圈子里，长得好看当然再好不过了。如果实在没那么好看，那么努力才是所有人都夺不走的。
